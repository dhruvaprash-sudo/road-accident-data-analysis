# Road Accidents Data Analysis System

This project is a menu-driven data analysis system built using Python, Pandas, NumPy, and Matplotlib to analyze road accident data.

## Features
- Data Visualization (Line Charts, Bar Graphs, Histogram)
- Data Analysis (Sorting & Filtering)
- CSV File Handling
- Data Manipulation (Insert/Delete Rows & Columns)

## Technologies Used
Python, Pandas, NumPy, Matplotlib
